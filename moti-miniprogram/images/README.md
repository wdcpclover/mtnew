# 图标资源说明

本目录需要放置以下图标文件（建议使用 48x48 或 96x96 的 PNG 格式）：

## 通用图标
- `logo.png` - 应用 Logo（闪电图标）
- `wechat.png` - 微信图标

## 导航图标
- `arrow-left.png` - 返回箭头
- `arrow-right.png` - 右箭头
- `arrow-down.png` - 下拉箭头

## 操作图标
- `search.png` - 搜索
- `message.png` - 消息
- `sun.png` - 太阳（浅色模式）
- `moon.png` - 月亮（深色模式）
- `pen.png` - 笔/编辑
- `close.png` - 关闭
- `share.png` - 分享
- `send.png` - 发送
- `bookmark.png` - 书签（未收藏）
- `bookmark-filled.png` - 书签（已收藏）

## TabBar 图标
- `tab-latest.png` - 最新（未选中）
- `tab-latest-active.png` - 最新（选中）
- `tab-topics.png` - 合集（未选中）
- `tab-topics-active.png` - 合集（选中）
- `tab-community.png` - 社区（未选中）
- `tab-community-active.png` - 社区（选中）
- `tab-profile.png` - 我的（未选中）
- `tab-profile-active.png` - 我的（选中）

## 功能图标
- `heart.png` - 点赞/喜欢
- `comment.png` - 评论
- `book.png` - 书籍
- `user.png` - 用户
- `users.png` - 多用户
- `star.png` - 星星/收藏
- `bell.png` - 通知铃铛
- `settings.png` - 设置
- `info.png` - 信息
- `credit-card.png` - 信用卡/付款
- `gift.png` - 礼物/兑换
- `calendar.png` - 日历
- `briefcase.png` - 公文包/产品
- `logout.png` - 退出
- `key.png` - 密钥
- `tag.png` - 标签
- `zap.png` - 闪电/能量

## 分类图标
- `category-0.png` 到 `category-5.png` - 分类图标

## 推荐图标资源
1. [Lucide Icons](https://lucide.dev/) - 开源图标库
2. [IconPark](https://iconpark.oceanengine.com/) - 字节跳动图标库
3. [Iconfont](https://www.iconfont.cn/) - 阿里巴巴图标库

## 使用建议
- 所有图标建议使用单色（#333333 或 #666666）
- 选中状态图标使用主题金色（#C99C00）
- 图标大小保持一致，建议 48x48px @2x
