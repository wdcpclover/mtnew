#### **一、 项目概述与目标**



本项目旨在为「莫提之地」打造一个集内容付费、知识分享与社群互动于一体的微信小程序平台。第一阶段的核心目标是实现项目启动和商业运营的最小闭环，涵盖用户从注册登录、浏览内容、付费解锁，到发帖、评论互动的完整流程。同时，建立一套功能完善的后台管理系统，支撑日常运营，并为二期规划的“朋友圈”、“资料库”及“AI Agent”等高级功能 奠定坚实的技术基础。



#### **二、 用户画像与角色权限**



系统将围绕以下核心用户角色进行设计：

- **未注册用户：** 可浏览部分公开内容，核心操作将引导其注册登录。
- **用户（已注册）：**
  - **未付费用户：** 享有基础的浏览和互动权限，但阅读深度内容受限，如每日最多阅读一篇帖子。
  - **付费用户：** 解锁所有付费内容和高级功能权限，如无限制阅读、发帖提问、内容划线做笔记 等。
- **管理员：** 拥有后台最高管理权限，负责内容审核、用户管理和社区运营。



#### **三、 功能范围清单 (第一阶段)**



| **模块**         | **功能项**       | **详细描述**                                                 |
| ---------------- | ---------------- | ------------------------------------------------------------ |
| **用户管理系统** | **注册与登录**   | 支持用户通过微信授权及手机验证码进行快速注册和登录。         |
|                  | **个人资料管理** | 用户可以自由设置与编辑自己的头像、昵称，并管理绑定的手机号。 |
|                  | **个人中心**     | 聚合展示用户的付费记录、已购产品有效期、自己收藏的帖子/评论/划线内容，以及点赞、评论、被点赞、被评论等互动记录。 |
|                  | **积分体系**     | 用户可通过互动或内容被点赞获得积分，并可在个人中心查看积分情况。 |
| **内容浏览**     | **帖子列表**     | 用户可浏览帖子列表，支持按最新、最热、最多收藏等维度排序，并可通过关键词搜索、时间或话题进行筛选。系统会自动区分帖子的已读和未读状态。 |
|                  | **阅读权限**     | 未付费用户每日最多可阅读一篇帖子内容，付费用户则可无限制点击阅读。 |
| **互动与创作**   | **帖子互动**     | 用户可以对帖子内容进行收藏、点赞、评论。                     |
|                  | **评论区互动**   | 用户可以对他人评论进行收藏、点赞、评论，并可对不良内容进行举报。 |
|                  | **提问发帖**     | 付费用户拥有发帖提问的权限。所有用户均可浏览全部提问帖，并对其进行投票或撤销投票。 |
|                  | **高级功能**     | 付费用户可在阅读帖子时，对部分内容进行划线并做笔记。         |
| **支付体系**     | **微信支付**     | 用户可调用微信支付接口付费，以开通特定功能和内容的访问权限。 |
| **管理员功能**   | **内容管理**     | 管理员可在后台对帖子进行编辑、添加、删除及置顶操作。         |
|                  | **评论管理**     | 管理员可对用户评论进行点赞、删除及置顶。                     |
|                  | **问答管理**     | 管理员可回答用户的提问帖，并可将优质问答从提问区转至精选问答列表。 |
